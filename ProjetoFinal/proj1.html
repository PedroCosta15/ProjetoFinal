Sistema Gestor de Tarefas

Os painéis de tarefas como o KanbanFlow são criados com vários recursos modulares que o tornam um ótimo projeto para brincar e aprender a programar.
Vai exegir um pouco de trabalho para começar, pois tu precisas de configurar uma UI que seja robusta e dinâmica.
Por sinal, este projeto é aconselhado ser feito com duas pessoas. Uma pessoa para trabalhar no front-end e outra pessoa para trabalhar no back-end.
Mas, não deixes que isso te desmotive!
Assim é uma chance para melhorares as tuas habilidades de comunicação e trabalho em equipa.
Irás ter de conversar sobre os projetos para garantir que ambos/as entendam completamente e saibam como os vossos módulos irão se conectar.
Isso é sempre mais desafiador do que parece.
