Conhecimentos

HTML

Tenho conhecimento sobre HTML, sei como criar "div" com os seus respetivos id ou class, sei organizar tudo de forma correta para que depois seja mais simples de aplicas estilo no site através do css.

CSS

Precebo bem de CSS, sei aplicar estilos nas respetivas áreas, formatar as "div", quando quero saber alguma propriedade pesquiso no google pois é complicado decorar todas as propriedades que existem.

Design

Gosto imenso de pensar em desins para sites, gosto de combinar as cores com as fotos de fundo e com os botões, e também de criar layouts novos.

Javascript

Precebo minimamente de java script, é uma linguagem que deixa os sites com mais funções, e mesmo não precebendo muito dessa linguagem, tenho sempre o google que tem a resposta para maior parte das duvidas, pois nós somos humanos e não comseguimos decorar tudo, por isso é que existem os computadores.

Boostrap

Boostrap é uma ferramenta muito util que faz com que consigamos poupar muito tempo quando estamos a desenvolver algum codigo para um site, é uma linguagem que conheço minimamente, já usei em alguns sites mas não é muito frequente usar.

GitHub

GitHub é uma "rede social" muito util para os programadores, e outros trabalhadores de area técnica, pois dá para armazenar ficheiros lá dentro do seu repositorio, e deixar tudo organizado, para não falar das ferramentas gratuitas que existem para estudantes, que por sinal ja me deram muito jeito.
